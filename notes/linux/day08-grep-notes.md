# grep Notes

## grep

Searches for matching text inside files.

Example

```bash
grep linux notes.txt
```

---

## grep -i

Performs a case-insensitive search.

```bash
grep -i linux notes.txt
```

Matches:

Linux

LINUX

linux

LiNuX

---

## Why grep is useful

Instead of opening very large files, grep searches only for the required information.

This makes searching faster and prevents unnecessary terminal output.
