# Day 8 Lab

## Objective

Practice searching text using grep.

---

## Tasks

- Search for a specific word
- Use grep -i
- Compare outputs
- Complete OverTheWire Bandit Level 7

---

## Commands Used

```bash
grep millionth data.txt

grep -i millionth data.txt
```

---

## Observation

grep prints only the matching lines instead of displaying the entire file.

Using grep is significantly more efficient than opening very large files with cat.
