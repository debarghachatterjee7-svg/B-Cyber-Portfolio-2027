# Linux Redirection Laboratory

## Objective

Practice Linux output streams and redirection techniques.

---

## Commands Practiced

```bash
cat fruits.txt
cat fruits.txt | sort
cat fruits.txt > output.txt
cat fruits.txt 2> output.txt
cat abc.txt 2> error.txt
find /home -name "*.txt"
find /home -name "*.xyz" 2> errors.txt
```

---

## Skills Learned

- Redirect standard output
- Redirect standard error
- Search recursively using find
- Store command output in files
- Verify generated files

--

## Outcome

Successfully understood how Linux separates output and errors into different streams and how each can be redirected independently.
