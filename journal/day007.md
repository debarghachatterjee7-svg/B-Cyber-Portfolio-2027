# 📚 Theory Learned

Today I learned how Linux can search files using much more than filenames. The `find` command allows searching based on ownership, file size, permissions and file type. This makes searching much more precise when working on large systems.

I also learned that Linux programs produce both normal output and error output. Using `2>/dev/null` redirects error messages to a special device that discards them, making the useful output easier to read.

---

# 💻 Commands Learned

```bash
find . -user username
find . -group groupname
find . -size 33c
find . -type f
find . -perm 644
find / -user bandit7 -group bandit6 -size 33c
find / -user bandit7 -group bandit6 -size 33c 2>/dev/null
cat /path/to/file
```
