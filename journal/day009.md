# 📚 Theory Learned

Today I learned one of Linux's most powerful concepts: the pipeline (`|`).

A pipeline connects commands together, allowing the output of one command to become the input of another.

I also learned how sorting and duplicate removal work together to process large amounts of text efficiently.

---

# 💻 Commands Practiced

## Pipe

```bash
command1 | command2
```

Purpose:

Passes the output of one command directly into another command.

---

## sort

```bash
sort filename
```

Purpose:

Sort lines alphabetically.

---

## Reverse Sort

```bash
sort -r filename
```

Purpose:

Sort lines in reverse alphabetical order.

---

## uniq

```bash
uniq filename
```

Purpose:

Removes consecutive duplicate lines.

---

## uniq -u

```bash
uniq -u
```

Purpose:

Displays only lines that appear exactly once.

---

## Combining Commands

```bash
sort filename | uniq
```

Purpose:

Sort data before removing duplicate lines.

---

```bash
sort filename | uniq -u
```

Purpose:

Display only unique lines after sorting.
