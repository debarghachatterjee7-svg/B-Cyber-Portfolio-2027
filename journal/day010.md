# Biggest Doubt

The output of:

```bash
wc notes.txt
```

initially confused me.

I saw:

```
4 4 36
```

and didn't understand what each number represented.

After learning more, I understood that `wc` always prints:

- Number of lines
- Number of words
- Number of bytes

The byte count also includes characters like newlines, which explains why it can be larger than the visible text length.
