# 🧠 Important Concepts Learned

Today I understood that:

- Public keys are safe to distribute.
- Private keys must never be shared.
- Fingerprints uniquely identify SSH keys.
- Randomart provides a visual representation of a fingerprint.
- SSH remembers trusted servers through `known_hosts`.
- Servers authorize users using `authorized_keys`.
- ED25519 is generally preferred for modern SSH deployments.
- RSA remains important for compatibility.
