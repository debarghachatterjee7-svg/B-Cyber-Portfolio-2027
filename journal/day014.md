# 📖 Theory Learned

Today's lesson introduced SSH authentication and Linux permission management. I learned that SSH allows secure communication between systems and can authenticate users using cryptographic key pairs instead of passwords. Linux protects important files through read, write, and execute permissions assigned to the owner, group, and others. I also learned that commands like `chmod 600` are commonly used to secure private key files by allowing access only to the owner.

While attempting OverTheWire, I realised that simply knowing commands is not enough. Understanding why those commands exist, how Linux internally handles authentication, and why specific permission values are required is much more important than remembering syntax alone. This understanding completely changed the way I want to approach future cybersecurity learning.
