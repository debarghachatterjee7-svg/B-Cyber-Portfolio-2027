
---

# 🧠 Biggest Learning

Linux treats normal output and error output as two completely different streams.

Errors are not mixed with normal output unless we explicitly combine them.

This makes debugging much easier.

---

# ❌ Mistakes Made

Initially expected error messages to appear inside the redirected output file.

After experimenting, understood that:

- `>` redirects only standard output.
- `2>` redirects only error messages.

Also verified the created files manually to understand what each contained.
